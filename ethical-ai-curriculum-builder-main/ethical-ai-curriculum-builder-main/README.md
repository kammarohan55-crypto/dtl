# Ethical AI Curriculum Builder

A simple, modular learning platform where students can choose a topic and understanding level to access structured learning modules, adaptive quizzes, flashcards, and responsible-use prompts.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI (Python)
- Data Storage: JSON

## Features
- Topic and level-based learning paths
- Modular content with industry insights
- Progress tracking
- Adaptive quizzes
- Flashcards for quick revision
- Responsible-use prompts integrated into learning
